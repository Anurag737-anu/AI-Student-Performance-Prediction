# 🎓 AI-Driven Student Performance Prediction System

## 📌 Project Overview
This project uses Machine Learning algorithms to predict student academic performance and analyze risk levels.

---

## 🚀 Features
- Student Score Prediction
- Grade Prediction
- Risk Analysis
- Data Visualization Dashboard
- Multiple ML Models Comparison

---

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

---

## 📊 Machine Learning Models
- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost

---

## 📈 Results
The project compares multiple models and selects the best-performing algorithm based on R2 Score.
